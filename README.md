# MAgentRender
an interactive pygame client for MAgent

**Usage**

```python
python render.py
```

**Requirement**

- python, version >= 3.5
- pygame, version >= 1.9.3

**features (will have)**

- PARALLEL RENDERING
- ASYNC RENDER WHILE TRAINING & SUPPORT TRAINING SUSPENSION
- REAL-TIME LEARNING CURVE
- ATTENTION: SELECT AN AGENT TO WATCH ITS LOCAL VIEW
- OTHERS
